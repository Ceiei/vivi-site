访问地址：<https://ceiei.github.io/vivi-site/>

这是我的个人网站，用来集中展示个人经历、项目作品、论文发表、获奖情况和学习博客。网站基于 Astro 构建，部署在 GitHub Pages，并支持从 Notion Database 自动同步博客内容。

## 网站内容

- **关于我**：个人介绍、教育背景、实习经历、技能标签、英语水平和联系方式。
- **项目汇总**：整理项目介绍、论文发表和获奖情况，部分项目提供独立详情页。
- **学习博客**：同步 Notion 中发布的学习笔记，按标签区分主题，适合记录 AI 学习、论文阅读、工具使用和实验复盘。

## 技术栈

- Astro
- Tailwind CSS
- GitHub Pages
- GitHub Actions
- Notion API

## 内容同步

博客文章维护在 Notion Database 中。GitHub Actions 会定时运行同步脚本，将已发布页面转换为 Markdown 并自动重新部署网站。
