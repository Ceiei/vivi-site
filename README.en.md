Website: <https://ceiei.github.io/vivi-site/>

This is my personal website for presenting my background, projects, publications, awards, and learning notes. The site is built with Astro, deployed on GitHub Pages, and supports automatic blog synchronization from a Notion Database.

## What Is Included

- **About Me**: personal introduction, education, internships, skills, English proficiency, and contact information.
- **Portfolio**: project summaries, publications, and awards, with dedicated detail pages for selected work.
- **Learning Blog**: notes synchronized from Notion, organized by tags for AI learning, paper reading, tool usage, and experiment reflections.

## Tech Stack

- Astro
- Tailwind CSS
- GitHub Pages
- GitHub Actions
- Notion API

## Content Sync

Blog posts are maintained in a Notion Database. GitHub Actions periodically runs the sync script, converts published Notion pages into Markdown files, and redeploys the website automatically.
